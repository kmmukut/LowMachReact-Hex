Evaluates physical properties for the entire domain.

If Cantera is enabled, it calls the C++ bridge to calculate mixture-averaged viscosity and diffusivity. Otherwise, it applies the uniform constant values defined in params.

@param mesh The computational mesh. @param params Simulation parameters and background conditions. @param Y Optional input mass fractions (ncells, nspecies). @param transport The property container to update.
