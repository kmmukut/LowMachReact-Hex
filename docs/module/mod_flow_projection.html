Incompressible Navier-Stokes solver using the Fractional-Step Projection method.

This module implements the core hydrodynamic solver for the LowMachReact-Hex framework. It solves the incompressible Navier-Stokes equations using a semi-implicit fractional-step approach:

  1. Predictor Step: An intermediate velocity $\mathbf{u}^*$ is calculated by advancing the momentum equation explicitly, excluding the new pressure gradient.

  2. Poisson Solve: A pressure correction potential $\phi = p^{n+1} - p^n$ is found by solving the Poisson equation derived from the continuity constraint ($\nabla \cdot \mathbf{u}^{n+1} = 0$).

  3. Corrector Step: The final velocity $\mathbf{u}^{n+1}$ and pressure $p^{n+1}$ are updated using the potential gradient.

The linear system for the Poisson equation is solved using a Preconditioned Conjugate Gradient (PCG) method with a diagonal (Jacobi) preconditioner.
