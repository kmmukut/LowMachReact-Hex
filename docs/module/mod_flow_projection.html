Incompressible Navier-Stokes solver using the projection method.

This module implements the fractional-step projection method for incompressible flows. It handles the momentum predictor step, pressure Poisson solve using Conjugate Gradient, and velocity correction. Compute distance from cell center to face center along normal.

@param mesh Full mesh. @param bc Boundary conditions. @param face_id Face ID. @param cell_id ID of the cell from which distance is measured. @param neighbor_id ID of the neighbor cell (used for periodic handling).
